How are graceful degradation and service workers related?  
Service workers in this website help the it to continue to operate even when it's not at its maximum network capabilities - this is related to graceful degradation because it allows the website to still work with reduced network capabilities. When the network fails, the website can still use service workers to restore cached recipes, allowing the users to browse it offline. Instead of crashing, the app remains mostly functional, which is an example of graceful degradation in action.
